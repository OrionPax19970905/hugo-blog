## 常用快捷键

### 数据库对象

```
数据库对象基本信息查看 : Ctrl+Q
修改表结构             : Ctrl+F6
查看 DDL               : Ctrl+B
搜索一切               : Double Shift
```

### SQL 编辑器

```
代码格式               : Ctrl+Alt+L
多位置编辑             : 按住 Alt 并拖动鼠标 / Shift+Alt+Click 放置游标的位置
实时模板               : Ctrl+J
智能代码块(子查询)选择  : Ctrl+W 扩大选中范围 / Ctrl+Shift+W 缩小选中范围
快速评估               : Alt+Click
生成 DDL               : Alt+Ins
重构表明               : Alt+Enter
查找表                 : Ctrl+N
```

### 数据编辑器

```
提交更改       : Ctrl+Enter
取消选中更改   : Ctrl+Alt+Z
全选           : Ctrl+A
文本搜索       : Ctrl+F
搜索并定位到列 : Ctrl+F12
```
